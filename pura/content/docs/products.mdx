Two products demonstrate the Pura mechanism. A third, vr.dev, handles verification independently and feeds evidence into Pura's completion receipt infrastructure.

## pura.xyz — Routing middleware dashboard

A consolidated dashboard that pulls live on-chain state from all deployed Pura contracts on Base Sepolia. Five sections in one app:

- **LLM gateway** — Routes calls across OpenAI and Anthropic based on which provider has capacity. Reads pool member units, completion rates, and dynamic pricing from BackpressurePool and PricingCurve. Chat interface with real API passthrough.
- **Nostr relay capacity** — Displays registered relay operators, their composite capacity scores, and anti-spam minimums across write/read/store pools. Reads from RelayCapacityRegistry and RelayPaymentPool.
- **Lightning routing** — Shows lightning node capacity (EMA-smoothed), routing fees, and active pool membership. Reads from LightningCapacityOracle and LightningRoutingPool.
- **Agent reputation** — Lists agents registered for a skill type, their capacity metrics, reputation scores (completions, slashes), and verified completions from CompletionTracker.
- **Simulator** — Runs the BPE allocation algorithm against live contract state with configurable parameters.

Includes one-click economy deployment (free tier + $9/mo hosted) and a deploy status tracker.

- Live: [pura.xyz](https://pura.xyz)
- Source: [pura/](https://github.com/puraxyz/puraxyz/tree/main/pura)

- Live: [bit.recipes](https://bit.recipes)
- Source: [bitrecipes/](https://github.com/puraxyz/puraxyz/tree/main/bitrecipes)

## vr.dev — Agent verification platform

A standalone verification layer for AI agent task outputs. 38 verifiers across 19 domains, organized in HARD/SOFT/AGENTIC tiers. A composition engine chains verifiers into pipelines, and an evidence chain produces SHA-256 content hashes anchored on-chain. When integrated with Pura, the evidence hash becomes the `taskId` in CompletionTracker receipts, binding reputation scores to tamper-evident proof of verified work.

- Live: [vr.dev](https://vr.dev)
- Source: [vr/](https://github.com/puraxyz/puraxyz/tree/main/vr)
- Python SDK: `pip install vrdev`
- Pricing: Free local, USDC micropayments via x402 for hosted verification
