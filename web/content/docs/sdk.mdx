The `@backproto/sdk` package provides a TypeScript client for interacting with all 17 Backproto [smart contracts](/docs/contracts) across 4 domains.

## Installation

```bash
npm install @backproto/sdk
```

## Features

- **Contract ABIs**: Pre-compiled ABIs for all 17 contracts
- **Type-safe interactions**: Full TypeScript types via [viem](https://viem.sh/)
- **13 action modules**: Core BPE + Demurrage + Relay + Lightning + Platform
- **Address registry**: Deployed contract addresses for supported networks
- **EIP-712 signing**: Helpers for attestations and completion receipts

## Usage

```typescript
import { addresses, contracts } from '@backproto/sdk'

// Get contract addresses for Base Sepolia
const { capacityRegistry, backpressurePool, stakeManager } = addresses.baseSepolia

// Use with viem
import { createPublicClient, http } from 'viem'
import { baseSepolia } from 'viem/chains'

const client = createPublicClient({
  chain: baseSepolia,
  transport: http(),
})

// Read smoothed capacity for a sink
const capacity = await client.readContract({
  ...contracts.capacityRegistry,
  functionName: 'getSmoothedCapacity',
  args: [taskTypeId, sinkAddress],
})
```

## Action Modules

### Core BPE

| Module | Description |
|--------|-------------|
| [`actions/sink`](https://github.com/backproto/backproto/blob/main/sdk/src/actions/sink.ts) | Register/deregister sinks |
| [`actions/source`](https://github.com/backproto/backproto/blob/main/sdk/src/actions/source.ts) | Register task types, start/stop payment streams |
| [`actions/pool`](https://github.com/backproto/backproto/blob/main/sdk/src/actions/pool.ts) | Create pools, rebalance, read member units |
| [`actions/stake`](https://github.com/backproto/backproto/blob/main/sdk/src/actions/stake.ts) | Stake/unstake tokens, read capacity cap |
| [`actions/buffer`](https://github.com/backproto/backproto/blob/main/sdk/src/actions/buffer.ts) | Deposit to / drain from escrow buffer |
| [`actions/pricing`](https://github.com/backproto/backproto/blob/main/sdk/src/actions/pricing.ts) | Report queue load, advance pricing epoch, read prices |
| [`actions/completion`](https://github.com/backproto/backproto/blob/main/sdk/src/actions/completion.ts) | Record completions, advance completion epoch |
| [`actions/aggregator`](https://github.com/backproto/backproto/blob/main/sdk/src/actions/aggregator.ts) | Submit batched off-chain attestations |
| [`signing`](https://github.com/backproto/backproto/blob/main/sdk/src/signing.ts) | EIP-712 signing helpers |

### Domain Extensions

| Module | Description |
|--------|-------------|
| [`actions/demurrage`](https://github.com/backproto/backproto/blob/main/sdk/src/actions/demurrage.ts) | Wrap/unwrap demurrage tokens, rebase, read decay rates |
| [`actions/relay`](https://github.com/backproto/backproto/blob/main/sdk/src/actions/relay.ts) | Register Nostr relays, join pools, set anti-spam minimums |
| [`actions/lightning`](https://github.com/backproto/backproto/blob/main/sdk/src/actions/lightning.ts) | Register Lightning nodes, join routing pools, get optimal routes |
| [`actions/platform`](https://github.com/backproto/backproto/blob/main/sdk/src/actions/platform.ts) | Aggregate reputation, stake discounts, route attestations |
